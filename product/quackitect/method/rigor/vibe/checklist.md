# Vibe — checklist template

The bare floor. **No human gates.** For throwaway or exploratory work, where speed beats
rigor. The plausibility check at `engage start` should push back if the vision sounds like
it will be kept or maintained. That is not a vibe project.

- **Intent** *(non-gate)*
  - [ ] one line: what and why

- **Build** *(non-gate)*
  - [ ] build it

- **Record** *(non-gate)*
  - [ ] one-line rationale of what was done — so even a throwaway leaves a trace

> No field loop. A throwaway has no field review. If you find yourself wanting one, it
> was not a vibe build — step up to lean.
