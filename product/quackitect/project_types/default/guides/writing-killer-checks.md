## Guide (load on demand)
A killer check is a step. It is dangerous to skip. It is easy to forget.

Rules:
- Keep the gated set short. Nine items or fewer.
- Word each as one checkable claim. Use the domain's own language.
- Place each at a real pause point.
- Revise a check whenever it misses something.

Killer checks are always human-adjudicated gates. You cannot tailor them away. You can only reword them.
